# Sample images

This folder is intentionally kept empty in the repository.

Create the following structure locally and add your own images:

demo/sample_images/
  real/{Architecture,Indoor,Nature,Objects}/
  ai/{Architecture,Indoor,Nature,Objects}/
  borderline/

Recommended sources:
- Real: Unsplash / Pexels
- AI: Lexica / PlaygroundAI / Civitai

Then run:
py -m src.batch_run
